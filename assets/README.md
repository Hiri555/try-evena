## ✅ Formats recommandés

- **JPG/JPEG** : Pour les photos
- **PNG** : Pour les images avec transparence
- **AVIF** : Format moderne et léger (optionnel)
- **WebP** : Alternative moderne (optionnel)

## 📐 Tailles optimales

- Images hero : 1200px de largeur minimum
- Images raisons : 800px de largeur minimum
- Images témoignages : 150x150px
- Image produit CTA : 400x400px

## 💡 Conseils

1. **Optimisation** : Compressez vos images pour un chargement rapide (utilisez TinyPNG ou ImageOptim)
2. **Nommage** : Respectez exactement les noms listés ci-dessus
3. **Qualité** : Privilégiez des images haute résolution pour un rendu professionnel
4. **Cohérence** : Utilisez un style visuel cohérent pour toutes les images
